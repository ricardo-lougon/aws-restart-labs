# AWS re/Start - Laboratórios Práticos

Bem-vindo ao repositório de laboratórios práticos realizados durante o programa **AWS re/Start**. Este repositório contém documentações e evidências técnicas de configurações em ambiente AWS, abrangendo desde fundamentos de computação até automação e machine learning.

## 📁 Estrutura do Repositório

Os laboratórios estão organizados por módulos temáticos para facilitar a consulta:

### 1. EC2, IAM e Armazenamento
* **EC2**: Bastion host e servidor web via CLI
* **Desafio EC2**: Implementação de VPC do zero
* **Troubleshooting**: Recuperação de pilha LAMP
* **IAM**: Gestão de grupos e permissões
* **AWS CLI**: Instalação e configuração
* **EBS**: Volumes, snapshots e restauração
* **Armazenamento**: Gestão via cron e S3 sync

### 2. S3
* **Site Estático**: Bucket, IAM e sincronização
* **Desafio S3**: Upload e permissões de acesso público
* **S3 Avançado**: Políticas e integração com SNS

### 3. Banco de Dados e Rede
* **RDS**: MySQL em configuração Multi-AZ
* **Migração**: Processo de migração com mysqldump
* **VPC**: Sub-redes, IGW e NAT Gateway
* **VPC Troubleshooting**: Análise de Flow Logs e ACLs

### 4. Computação Avançada
* **Systems Manager**: Automação de instâncias
* **ELB + Auto Scaling**: Alta disponibilidade
* **Auto Scaling CLI**: Gestão via linha de comando
* **Route 53**: Estratégias de failover DNS

### 5. Serverless e Notificações
* **Lambda**: DataExtractor com EventBridge
* **Desafio Lambda**: Contador de palavras

### 6. Monitoramento, Governança e Segurança
* **CloudWatch**: Logs, Events, Agent e Config
* **CloudTrail**: Investigação de incidentes
* **Tags**: Automação com JMESPath e Stopinator

### 7. Otimização e IaC
* **Otimização**: Rightsizing e uso do Pricing Calculator
* **CloudFormation**: Ciclo de vida e YAML
* **CFN Troubleshooting**: WaitCondition e detecção de drift
* **Desafio CFN**: Automação de VPC + EC2

### 8. Machine Learning
* **SageMaker**: Implementação do algoritmo XGBoost
